# Instruções para Executar a Aplicação

1. Certifique-se de ter o Docker e o Docker Compose instalados.

2. No diretório do projeto, execute o comando:
   docker-compose up -d

3. Acesse a URL da aplicação:
   http://localhost:3000/consulta-dados

4. O link do GitHub é https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO
   O link do DockerHub é https://hub.docker.com/repository/docker/SEU_USUARIO/NOME_DO_REPOSITORIO
